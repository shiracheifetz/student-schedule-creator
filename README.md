# student-schedule-creator
# Authors: Shira Cheifetz, Leeba Cohen, and Dassi Berry
# Date Due: 3/24/2026
# Description: Student Schedule is a customizable scheduling website designed to help students organize their academic timetable efficiently. The application allows users to add, edit, and delete courses so their schedule always reflects their current classes. It automatically detects and notifies students when course times overlap, helping prevent conflicts. All courses are displayed in a clear, organized weekly calendar view, making it easy for students to see their weekly classes at a glance and manage their time more effectively.
